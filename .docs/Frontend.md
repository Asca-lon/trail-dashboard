# 개요

기상-철도 리스크 의사결정 지원 시스템의 프론트엔드 문서 인덱스입니다.

기존 통합 문서는 페이지별 문서로 분리했으며, 새로운 기록은 해당 페이지 문서에서 관리합니다.

# 구현 목적

- 필요한 페이지 기록을 빠르게 찾습니다.
- 서로 다른 페이지의 변경 이력이 한 파일에 누적되는 문제를 방지합니다.
- 페이지별 책임과 공통 기능의 경계를 명확하게 유지합니다.

# 코드 설명

문서만 재구성했으며 프론트엔드 애플리케이션 코드는 변경하지 않았습니다. 기존 작업 기록은 제목 단위로 분류하여 관련 문서로 이동했습니다.

# API

API 변경 사항은 없습니다. 페이지별 API 및 Mock 데이터 설명은 각 문서에서 확인합니다.

# Database

Database 변경 사항은 없습니다.

# 주의사항

- 공통 기능과 두 페이지 이상에 영향을 주는 기록은 `Frontend_Common.md`에서 관리합니다.
- 새 기록을 추가할 때 같은 내용을 여러 페이지 문서에 반복하지 않습니다.
- 기존 `.docs/Dashboard_Change_Spec.md`는 별도 변경 명세이므로 유지합니다.
